文档自动保存系统使用指南

1. 系统概述

这是一个用于快速保存和管理各类文档的系统，支持以下四个分类：

  • 工作文档 (work)
  • 学习笔记 (study)
  • 生活记录 (life)
  • 兴趣爱好 (hobby)

3. 使用流程

  1. 复制内容

    • 选中要保存的内容
    • 使用 Cmd+C 复制到剪贴板

  2. 执行保存

    • 根据内容类型，使用对应的保存命令
    • 系统会自动：
      • 创建带时间戳的 markdown 文件
      • 添加文件到 Git
      • 进行本地提交

  3. 推送更改

    • 使用 pushall 命令推送所有更改
    • 命令会自动：
      • 添加所有更改 (git add .)
      • 创建提交 (git commit)
      • 推送到远程 (git push)

  • save.sh: 命令行接口，支持分类参数

5.2 保存格式

每个保存的文件都包含：
